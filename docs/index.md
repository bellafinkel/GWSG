## Installation
{:.mt-lg-0}

### git
  Simply clone the repository and run import the module
  
### pip
  ```bash
  pip install gwsignalgenerator
  ```

### GitHub Pages

And add this line to your Jekyll site's `_config.yml`:

```yaml
remote_theme: allejo/jekyll-docs-theme@v{{ site.project.version }}
```
